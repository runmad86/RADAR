# OutputResolver

## Rolle
OutputResolver vaelger de faa naturlige outputs ud fra kampens profil.

## Foerste input
- primary profile
- profile strength
- signal-stoette
- matchup support

## Foerste principper
- Open -> BTTS / over 2,5 som base
- Hot -> over kort / begge hold kort som base
- Locked -> under 2,5 som base
- None -> ingen klar case
- Pressure senere -> favoritsejr / underdog scorer ikke som base

## Fast regel
En kamp boer typisk ende i 1-2 base-cases og eventuelt 1 spike-case. Ikke mere.

## Vigtig regel
OutputResolver maa foelge profilens hovedlugt. Den maa ikke selv begynde at opfinde en ny profil bagom motoren.
